  • Angular is a front-end web application framework led by a team at Google. It was orginally called, AngularJS, but was rebranded after major changes were made.
  • AngularJS focused on Controllers and Scope, is slower, and wasn't built with mobile support.
  • Angular is sometimes referred to as Angular2+. Angular was completely rewritten by the team at Google. It follows component-based UI and there are no more controllers and scope. Components replace controllers. It is faster, was built with mobile support, and is compatible with TypeScript, ES5, and ES6.
