What am I using?
  • TypeScript is an open-source programming language maintained by Microsoft. It is a decendent of the JavaScript language that allows for optional static typing.
  • It offers additional features to ECMAScript5 5 such as: Type annotations, compile-time checking, Type inference, Type Erasure, Interfaces, and more.
  • Angular is a front-end web application framework led by a team at Google. It was orginally called, AngularJS, but was rebranded after major changes were made.
  • AngularJS focused on Controllers and Scope, is slower, and wasn't built with mobile support.
  • Angular is sometimes referred to as Angular2+. Angular was completely rewritten by the team at Google. It follows component-based UI and there are no more controllers and scope. Components replace controllers. It is faster, was built with mobile support, and is compatible with TypeScript, ES5, and ES6.
  • Firebase is a mobile and web application development platform acquired by Google in 2014. It offers anayltics, cloud messaging, authorization on the client side, real time database, hosting, testing lab for Android, crash reporting and many other services for free.
  • I am specifially using it for their NOSQL realtime data base and authorization service but I plan on using more of the services when I start learning mobile development.
  • Npm is a package manager for JavaScript. It is the default package manager for Node.js. Available packages can be found on the npm website. It is written entirely in JavaScript.
  • Npm can manage packages that are local dependencies of a single project, and also globally install JavaScript tools. Dependencies of a project are stored in the package.json file and dependencies can specify a version range to help avoid unwanted breakage. You can also tag your package with a particular version that you know works.
  • Auth0 is a service created by developers that creates a standard for authorization and authentication. You can connect any application to Auth0 and define the identity providers you want to use.
